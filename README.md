# Escape-The-Death
YouTube：https://www.youtube.com/watch?v=PEOdr92A2aU&list=PLfq8CNJ5gGBqhNTHXhe4sgSBFZyy5NF0V&index=7

AI-driven characters / 3D Puzzle / Engine: Unity

This game design project revolves around conversations with AI-driven characters generated by a ChatGPT model. Through dialogue, players uncover the story of individuals with Dissociative Identity Disorder, hidden within a dreamlike narrative. Player choices shape unique endings, exploring the dual challenges of disability and societal pressure. The project highlights resilience and identity while showcasing AI's potential for immersive storytelling.
